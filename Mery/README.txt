# Mery

Mery は、テキスト編集を簡単で楽しいものに変えるシンプルなテキストエディターです。小説の執筆からプログラムのコーディングまで、どんなテキストファイルも手軽に編集できます。

## 機能

- シンプルなデザイン: 初心者から上級者まで、誰でも使いやすい設計で、必要な機能はすぐに見つかります。
- 色分け表示: テキストの編集モードを切り替えると、テキストが見やすくなり、プログラムのコードもカラフルに編集できます。
- 文字サポート: ユニコード、サロゲートペア、異体字シーケンス (IVS) に対応し、言語や文字に制限がありません。
- カラー絵文字: Windows 8.1 以降のカラー絵文字にも対応しています。
- 美しい文字描画: DirectWrite により、文字が美しく鮮明に表示され、読みやすさを提供します。
- トゥルーインライン入力: トゥルーインラインの日本語入力方式に対応し、変換位置が一目でわかります。
- 縦書き編集: 日本語の文章や小説を書くのに最適で、文字を縦方向に表示して編集できます。
- マーカー機能: 重要な部分に色を付けて文書の要点を把握しやすくします。
- 正規表現検索/置換: 鬼雲、鬼車を駆使した強力な正規表現検索/置換機能を備え、テキストの効率的な編集が可能です。
- タブ機能: 複数の文書を同時に開き、タブを使って簡単に切り替えられます。
- マクロ機能: 操作を記録して自動実行でき、作業効率を向上させることができます。また、JavaScript でマクロを記述することも可能です。
- 単語補完: 入力された文字から始まる単語をリストから選択して、入力ミスを最小限に抑えます。
- アウトライン機能: 文書の見出しを階層的に表示して、文書全体の構造を把握しやすくします。
- 印刷機能: 編集内容に応じたカラフルな印刷や段組み印刷がサポートされており、文書の印刷品質を向上させます。
- HTML 整形: HTML Tidy を使用した文書整形とエラーチェックで、HTML の品質を向上させます。
- Zen モード: エディター部分のみを全画面表示して、作業に完全に集中できます。
- タイプライター スクロール: 編集箇所を画面中央に配置し、視線移動を最小限に抑え、快適な編集をサポートします。
- スクロール バー マーカー: スクロール バーに検索結果や変更箇所を表示し、文書の状態を簡単に確認できます。
- 自動保存: 強制終了や保存し忘れに備えて、文書を自動で保存します。
- ワークスペース: 終了時の状態を記憶し、作業を中断した場所から再開できます。
- マルチ カーソル: 複数箇所を同時に編集でき、一括変更や整列などが簡単に行えます。
- あいまい検索: 類似した文字列を簡単に検索し、表記のゆれや誤字などにも対応できます。

## はじめに

**動作環境**
- Windows 11、10、8.1、8、7、Vista、XP をサポートしています。

**アーカイブに含まれるファイル**
- Mery.exe: アプリケーション本体
- merycfg.dll: 編集モード ライブラリ
- meryres.dll: リソース ライブラリ
- onig.dll: 鬼車正規表現ライブラリ
- onigmo.dll: 鬼雲正規表現ライブラリ
- migemo.dll: C/Migemo ライブラリ
- tidy.dll: HTML 整形ライブラリ
- libhunspell.dll: スペル チェック ライブラリ
- uchardet.dll: エンコード検出ライブラリ
- qjs.dll: QuickJS ライブラリ
- WebView2Loader.dll: WebView2 ライブラリ
- Macros/: マクロ フォルダー (サンプルマクロを含む)
- Plugins/: プラグイン フォルダー (サンプルプラグインを含む)
- Dicts/: スペル チェック辞書フォルダー
- Ext/: 単語補完辞書フォルダー
- dict/: Migemo 辞書フォルダー
- ThirdPartyNotices.txt: サードパーティ製コンポーネントのライセンス
- eula.html: ソフトウェア使用許諾書
- README.txt: 本ドキュメント

## インストール

- インストーラー版をご利用の場合は、インストーラーを実行し、表示されるインストール ウィザードに従ってインストールしてください。
- ZIP 版をご利用の場合は、アーカイブを展開し、選択したフォルダーに配置してください。

## アップデート

- インストーラー版をご利用の場合は、インストーラーを実行し、表示されるインストール ウィザードに従ってアップデートしてください。
- ポータブル版、または ZIP 版をご利用の場合は、アーカイブを展開し、既存のファイルを上書きしてください。
- このとき、アーカイブを展開したフォルダーでは、Mery.exe を実行しないでください。
- `Macros`、`Plugins`、`Dicts`、`Ext` フォルダー内のコンテンツが上書きされますので、必要に応じてバックアップを取ることをおすすめします。

## アンインストール

- インストーラー版をご利用の場合は、[アプリと機能] またはコントロール パネルの [プログラムと機能] からプログラムをアンインストールしてください。
- ZIP 版をご利用の場合は、フォルダーごとごみ箱に移動してください。
- 設定ファイルは以下のフォルダーに保存されているため、手動で削除してください。

**Windows 11、10、8.1、8、7、Vista**
```
C:\Users\<UserName>\AppData\Roaming\Mery
```

**Windows XP**
```
C:\Documents and Settings\<UserName>\Application Data\Mery
```
